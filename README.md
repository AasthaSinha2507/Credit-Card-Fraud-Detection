# Credit Card Fraud Detection

## 📌 Project Overview
This project detects fraudulent credit card transactions using machine learning techniques.

## 🔧 Tools & Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib & Seaborn

## 🚀 Models Used
- Logistic Regression
- Random Forest

## 📈 Key Insights
- Dataset is highly imbalanced
- Random Forest performs better
- Recall is important for fraud detection

## ▶️ How to Run
1. Upload dataset in Google Colab
2. Run all cells
